# Personal Website

My personal portfolio website — a single-page React app showcasing my experience, projects, skills, and contact info.

## ✨ Features

- **Hero** — Introduction and headline section
- **Experience** — Work history and professional background
- **Projects** — Highlighted personal and professional projects
- **Skills** — Technologies and tools I work with
- **Contact** — Ways to get in touch

## 🛠 Tech Stack

- [React](https://react.dev/) (v19)
- [Tailwind CSS](https://tailwindcss.com/) (v3)
- Bootstrapped with [Create React App](https://github.com/facebook/create-react-app)

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm start
```

The app will be available at [http://localhost:3000](http://localhost:3000).

## 📦 Build for Production

```bash
npm run build
```

Outputs an optimized production build to the `build/` folder.
